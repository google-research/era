This repository contains the code associated with the paper:
**[An AI system to help scientists write expert-level empirical software](https://arxiv.org/abs/2509.06503)** (Aygun, et al., 2025).

## Overview
ERA (Empirical Research Assistant) is an AI system designed to assist scientists in writing high-quality empirical software.

## Implementation: Flat UCB Tree Search (FUTS)
The `implementation/` directory contains a generic single-threaded reference implementation of the Tree Search algorithm described in Algorithm 1 of the paper.

The search function in `implementation/futs.py` requires two user-provided functions:
- `generate_fn`: Generates a new solution from a past solution.
- - `execute_fn`: Scores a candidate solution in the context of the problem.
 
  - The search iterates `num_iterations` times, expanding a search tree one node at a time using the Flat UCB algorithm, and returns the best solution found.
